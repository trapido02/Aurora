## How to build
#### Windows
First make sure you have at least CMake version 3.20 installed.
1. Clone the git respsitory with `git clone https://github.com/trapido02/Aurora`
2. Create a new directory inside the cloned repository where you will build the project (called "build" or something)
3. Open that directory in cmd and run `cmake ../`

That should generate the visual studio solution files.

Alternatively after you cloned the repository you can left click inside the directory and choose Open with Visual Studio.
That should automatically build the project and you can just run it from there.
